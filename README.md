# Empresa Multiservicios Ivyst - Website
* [Ver el sitio en vivo aqui] https://leondev0.github.io/Ivyst/
Este es el repositorio oficial del sitio web desarrollado para la **Empresa Multiservicios Ivyst**. El proyecto consistió en el diseño, desarrollo y despliegue de una plataforma web profesional para mejorar la presencia digital y la atención al cliente de la empresa.

## Características
* [cite_start]**Interfaz Dinámica:** Diseño responsivo y moderno centrado en la experiencia del usuario (UX)[cite: 54].
* [cite_start]**Optimización de Procesos:** Integración de soluciones tecnológicas para agilizar la consulta de servicios[cite: 24].
* **Código Limpio:** Estructura de archivos organizada y optimizada para facilitar el mantenimiento futuro.

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica del contenido.
* **CSS3:** Estilizacion avanzada y diseño responsivo.
* [cite_start]**JavaScript:** Lógica de interacción y funcionalidades dinámicas.

## Estructura del Proyecto
* `/css`: Hojas de estilo.
* `/js`: Scripts de JavaScript.
* `/assets`: Imágenes y recursos gráficos.
* `index.html`: Punto de entrada principal.
